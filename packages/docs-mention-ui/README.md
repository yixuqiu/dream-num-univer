# @univerjs/docs-mention-ui

## Introduction

`@univerjs/docs-mention-ui` package provides the Mention function for Docs.

## Usage

### Installation

```shell
# Using npm
npm install @univerjs/docs-mention-ui

# Using pnpm
pnpm add @univerjs/docs-mention-ui
```
