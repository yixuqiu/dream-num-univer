## Introduction

The `@univerjs/network` library provides basic network service capabilities, primarily for use in collaborative editing scenarios.

## Usage

### Installation

```shell
# Using npm
npm install @univerjs/network

# Using pnpm
pnpm add @univerjs/network
```
