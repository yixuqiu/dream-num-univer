## Introduction

`@univerjs/sheets-filter` is a filter plugin for Univer Sheet.

## Usage

You should use this plugin with the `@univerjs/sheets-filter-ui` package.

### Installation

```shell
# Using npm
npm install @univerjs/sheets-filter

# Using pnpm
pnpm add @univerjs/sheets-filter
```
